# Privacy Policy for Paul Graham Essay Viewer

**Last Updated: January 17, 2026**

## Overview

Paul Graham Essay Viewer ("the Extension") is committed to protecting your privacy. This privacy policy explains our data collection and usage practices.

## Data Collection

**We do not collect, store, or transmit any user data.**

The Extension:
- ✅ Does NOT collect personal information
- ✅ Does NOT track browsing history
- ✅ Does NOT use cookies or local storage
- ✅ Does NOT make external network requests
- ✅ Does NOT share data with third parties
- ✅ Does NOT use analytics or tracking tools

## How the Extension Works

The Extension operates entirely locally within your browser:

1. **Visual Modifications Only**: The Extension modifies the visual presentation of paulgraham.com by injecting CSS styles and reorganizing DOM elements
2. **Local Processing**: All code execution happens locally in your browser
3. **No Data Transmission**: No information is sent to external servers
4. **Domain-Specific**: The Extension only operates on paulgraham.com

## Permissions Explanation

The Extension requires the following permission:

### Host Permission (`https://www.paulgraham.com/*`)
- **Purpose**: To inject CSS styles and modify the DOM structure for improved readability
- **Usage**: Only used to enhance the visual presentation of paulgraham.com
- **Data Access**: The Extension reads the page structure to extract navigation elements but does not collect, store, or transmit this information

## Third-Party Services

The Extension does not integrate with or use any third-party services, APIs, or analytics platforms.

## Data Security

Since the Extension does not collect any data, there is no user data to secure or protect. All Extension code runs locally within your browser.

## Children's Privacy

The Extension does not knowingly collect information from anyone, including children under the age of 13.
